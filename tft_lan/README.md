# ⚔ TFT 시즌3 갤럭시 — LAN 오토배틀러

브라우저 기반 TFT(팀파이트 택틱스) 시즌3 갤럭시 스타일 오토배틀러입니다.  
같은 LAN에 연결된 최대 4명이 웹 브라우저로 대전할 수 있습니다.

## 실행 방법

```bash
# 1. 의존성 설치
pip install aiohttp

# 2. 서버 실행
cd tft_lan
python server.py
```

서버가 실행되면 **로컬 IP와 포트**가 출력됩니다.

```
⚔  TFT 시즌3 갤럭시 - LAN 오토배틀러  ⚔
브라우저에서 접속: http://192.168.x.x:8080
같은 PC:         http://localhost:8080
```

## 접속

- 같은 PC: `http://localhost:8080`
- 같은 네트워크 (LAN): `http://<서버IP>:8080`

## 조작법

| 동작 | 방법 |
|------|------|
| 챔피언 구매 | 상점 카드 클릭 |
| 배치 | 벤치에서 보드로 **드래그 앤 드롭** |
| 위치 이동 | 보드 셀 → 보드 셀 드래그 |
| 회수 | 보드 → 벤치 드래그 |
| 판매 | 벤치/보드 유닛 **우클릭** |
| 아이템 장착 | 오른쪽 아이템 목록 클릭 → 유닛 클릭 |
| 새로고침 | 🔄 버튼 (2골드) |
| 레벨업 | ⬆ 버튼 (4 XP) |
| 준비 완료 | ✅ 버튼 |

## 게임 규칙

- **라운드 1~3**: PvE (크리프 전) — 아이템 획득
- **라운드 4+**: PvP 대전
- **증강**: 라운드 4, 12, 20에 3가지 증강 중 선택
- **레벨 9**까지 성장 가능, 5코스트 챔피언 등장
- **동일 챔피언 3개** → ★2 승급, ★2 3개 → ★3 승급
- **아이템 조합**: 기본 아이템 2개가 합쳐져 완성 아이템이 됩니다
- **시너지**: 같은 특성 챔피언을 배치하면 시너지 발동
- **준비 시간**: 30초 (전원 준비 완료 시 즉시 전투)
- 체력이 0이 되면 탈락, 마지막 생존자가 승리

## 시스템 요구

- Python 3.8+
- aiohttp 3.9+
- 모던 브라우저 (Chrome/Edge/Firefox 등)
